@AGENTS.md

This is a web UI admin interface for [open-oscar-server](https://github.com/mk6i/open-oscar-server). It is built with Tailwind, NextJS 16, and, of course, open-oscar-server. No auth is required since this will run local to the server and be locked away inside private network space.

Use the OpenAPI docs in docs/api.yml and make a nice admin UI for managing the server.
